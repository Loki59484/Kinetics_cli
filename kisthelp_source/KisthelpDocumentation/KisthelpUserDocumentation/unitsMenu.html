III.5 Description of the Units menu
 
 
 

1) kJ / mol (default) :

        This option gives the possibility to display the Energy Units in kJ.
 
 
 
 

2) T  :

        This option gives the possibility to change the current temperature unit (K, °C, °F).

        Warning: for instance, setting the current temperature unit to °C will change the temperature, for instance 298, from 298K to 298°C but not to 25°C.

3) P  :

        This option gives the possibility to set the pressure unit (Pa, Bar, Torr, Atm).

        Warning: for instance, setting the pressure unit to Torr, will change the pressure, for instance 1, from 1 bar to 1 Torr but not to 750.06 Torrs.
